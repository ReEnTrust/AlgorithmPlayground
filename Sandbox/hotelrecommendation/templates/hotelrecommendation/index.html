Algorithm Playground

An experimental platform to explore recommender systems

Introduction

Welcome to the Algorithm Playground! This experimental tool is about making algorithms more trustworthy. It aims at bringing a human perspective on decisions made by recommender systems.

This is a platform developed by the Edinburgh Team involved in the ReEnTrust Project. This project is about how platform could behave to rebuild user trust. The philosophy of this project is to provide tools for a responsible and transparent using of data and algorithms.

This platform gives you the chance to discover how the algorithms influence your choice when looking for a hotel to book. You can compute lists of recommended hotels for several users with different genders and traits. As an outside observer, you can play with the inputs and watch the changings in the outcome. You have the possibility to switch between several recommendation algorithms to see how this choice is crucial for any recommender system.

How it works?

The Algorithm Playground makes you explore different versions of a recommendation algorithm. It is based on a so-called hybrid system to specifically recommend hotels. Such systems are based on the ratings from former users to determine a quality score for each hotel. The given recommendation is computed in two steps. First, a form is proposed to be filled, aiming at giving personal details about a fictional customer. Thus, the platform has enough information to search, in the database, which former user is the most similar. Then, the finale result is a recommendation based on the similar user ratings, which are stored in a database consulted by the algorithm to compute which hotels should be recommended to any fictional customer similar to this user.

What is this tool?

Algorithm Playground is a tool that allows to play with a hybrid recommencer system for hotel recommendation, by using personal details. The filled profile will be compared with thousands of other users. A database of ratings and hotels is used to compute the most accurate recommendation.
